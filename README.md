# Lasles Vpn 🔐

A sleek and modern Vpn website built with Vite.

## 🚀 Features
- Fast and lightweight Vite setup
- Responsive design
- Easy deployment with Vercel

## 🛠️ Tech Stack
- Vite
- HTML/CSS/JavaScript
- TailwindCss
- Supabase Auth
- GitHub + Vercel
